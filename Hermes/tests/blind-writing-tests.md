# Gate 2 — Blind writing tests (Alyssa's definition of done)

Writer = fresh session, loads only the README recipe for the format. Judge = separate fresh session, loads only TASTE.md + examples/ (good + bad). Judge never sees which output is generated where a lineup applies.

## The four acceptance tests

1. **Write a carousel** — seed: "cold email lists hurt deliverability" (Week 1 Email, Monday slot). Deliver slide-by-slide text + caption + correct comment word. Judge scores rubric + slot rules.
2. **Write a Simmer email** — seed: any real domestic micro-scene mapped to one marketing lesson. Full email incl. subject + PS.
3. **Email → carousel** — input: a real Soft Sunday from the vault (e.g., if-lorelai-had-an-online-biz.md). Apply craft/email-to-carousel.md. Judge compares against 2 real captions from examples/good/captions.md.
4. **Pitch a product Alyssa would actually sell** — propose ONE new offer that improves the suite: name (her grammar), price position (justified against the ladder), theme lane, format, one-liner, "who it's for," elevator pitch in her voice, and what it must NOT be. Judge checks against craft/offer-development.md hard stops + BUSINESS.md (no duplication, no invented facts).

## Supporting probes
5. 10 subject lines + preview text for a stated email (mix of registers, one per pattern max). 6. A sales-page pain section for List Resuscitator. 7. A Soft Sunday Pep Talk opening. 8. A teaching outline for a hypothetical lesson "batching Pinterest pins" in her lesson arc.

## Scoring (per output)
- TASTE.md rubric (0-2 × 9 axes). Kill: any 0 on refusals/pain-framing/CTA; pass ≥14/18.
- 8 red-flag checklist — any hit = fail.
- Golden side-by-side: rhythm, humor placement, metaphor altitude, specificity vs. 2 real passages.
- **Lineup test** (tests 1-3): hide 1 generated passage among 3 real ones; fresh judge picks the fake; repeat 3×. Pass = judge wrong or unsure in ≥2 of 3 rounds.

## Logging
Each run → tests/results/YYYY-MM-DD-gate2.md: outputs, scores, failures, fixes applied, rerun outcome. Rerun after every SOUL/STYLE/TASTE edit.
