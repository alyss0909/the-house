# Carousel Interconnection Rule

## Trigger

Alyssa clarified that carousel analysis is not useful unless each post can be understood as a connected whole: slides, captions, metrics, themes, content rail, and product/offer context must all be linked together.

## Insight

The carousel slide-by-slide breakdown must not be a standalone analysis layer. Every post section needs:

- a wikilink to the whole slide folder
- a wikilink to the caption/metrics source
- explicit Theme, Post type / rail, and Product / offer connection fields
- a wikilink to each individual slide image under the matching slide breakdown
- direct product or program links when a slide references an offer, waitlist, product, or program
