# Monthly Reviews

Monthly wrap-up documents synthesizing weekly wraps for a given calendar month. Part of the review pyramid: close-session (per session) → weekly review (per week) → monthly review (per month).

## Purpose

The monthly review is intentionally light. The weekly tier already extracted decisions, insights, and open threads from raw session logs — the monthly layer reads weekly wraps only and focuses on:

1. **Bigger picture narrative** — what defined the month, what shifted.
2. **Pattern recognition** — recurring themes, persistent blockers, momentum.
3. **Insight graduation** — which session/weekly insights are now permanent rules (ready for SOPs, Guidelines, or Workstreams).
4. **Next month setup** — what to carry forward and prioritize.

## How to generate

Run `/monthly-review` at any point during or after a month. Larry reads all weekly wraps for the target month and writes the wrap doc here.

- Default: current calendar month
- Override: `/monthly-review --month 2026-05`

## Naming convention

`YYYY/YYYY-MM-monthly-wrap.md`

Example: `2026/2026-05-monthly-wrap.md`
