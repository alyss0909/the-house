# Notebook Sources Prompt

A prompt for getting a structured breakdown of every source in a notebook or document collection. Useful when importing a new body of material and wanting a quick map of what's in it.

```
List every source in this notebook. For each one, give me the title, its main topic, three key points it covers, and how it relates to the other sources.
```
