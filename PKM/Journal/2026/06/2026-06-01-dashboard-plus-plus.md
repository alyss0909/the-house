# Dashboard++

A simple Obsidian vault organization method by TfTHacker. Two ingredients: a CSS snippet + markdown lists. In Reading view, lists render as columns — creating a home dashboard that works like a personal Wikipedia.

**The idea:** a living home page that groups notes by topic, links into subtopic dashboards, links into individual notes. Structure emerges from use — you don't impose it upfront.

**What it takes to set up:**
1. Drop the [Dashboard++ CSS snippet](https://github.com/TfTHacker/DashboardPlusPlus/blob/master/.obsidian/snippets/dashboard.css) into Obsidian snippets folder and enable it
2. Create a home page file, add `cssclass: dashboard` to frontmatter
3. Use H1 headings as topic groups, markdown lists for the links beneath each

Optional extras: Banners plugin (header images), Homepage plugin (auto-open on launch), Dataview (recently updated files widget).

Sample vault to play with: https://github.com/TfTHacker/DashboardPlusPlus
