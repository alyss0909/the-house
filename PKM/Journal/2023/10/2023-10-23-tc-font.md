# TC font
For elements where the font cannot be changed directly, you do need CSS. You can also just add it to the checkout page box in the custom code section under Tracking.
(This uses the Satisfy font as an example.)
<style>
.thrivecart-body {
  font-family: "Satisfy";
}
</style>
