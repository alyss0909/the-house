# My Life - Index

The five life concepts. One file per concept. The wiki connects them via `[[wikilinks]]`.

## Subsections

- **Topics/** - interest areas and reference themes.
- **Habits/** - recurring rhythms.
- **Goals/** - what you are moving toward.
- **Projects/** - bounded efforts with a finish line.
- **Key Elements/** - stable top-level domains (Business, Life) and their children.
- **Offers/** - products, programs, and services.

## Active files

Key Elements, Topics, Goals, Projects, Habits, and Offers each have their own `INDEX.md`. See those for the current file list. Scaffold course samples have been removed.

## Naming

Files in My Life use kebab-case slugs. No date prefix; these are concept files, not date-driven.

See [[GL-001-file-naming-conventions]].

## Cross-links

- Journal entries reference these concepts via `[[wikilinks]]`.
- A Goal can reference the Project that delivers it; a Project can reference the Habits that sustain it.
- The Topic file is the canonical home for the topic. Anywhere the topic comes up, link here. Do not restate the topic's definition inside a Journal entry.
