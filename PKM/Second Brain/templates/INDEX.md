## How to build a template

1. Find the best real example in `examples/`
2. Ask Larry: "Turn this into a reusable template"
3. Larry extracts the structure, replaces specifics with [brackets], preserves voice
4. Review — does it still sound like Alyssa?
5. Save here, update status above

## How a team member uses this

Pick the template for the job → fill in [brackets] → adjust for the specific offer, audience, and moment → cross-check against `analysis/writing-style.md` before sending.
