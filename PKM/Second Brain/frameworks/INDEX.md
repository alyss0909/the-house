## How a team member uses this

**Daily email concept:** Check `hook-vault.md` for an opening hook → check `rto-vault.md` for offer timing → go to `examples/simmers/` for a real simmer to pair with → write

**Daily social:** Check `hook-vault.md` → go to `programs/` or `examples/carousels/` for content to draw from → apply framework to the format needed

**Sequence planning:** Start with `sequence-prompts.md` → pair with `segmentation-vault.md` → close every email with a P.S. from `ps-vault.md`
