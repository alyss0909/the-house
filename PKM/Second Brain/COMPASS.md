## The One Rule

**Load only what the task requires. Every file you load costs tokens before a word is written.**

Most tasks need `voice-profile.md` + 1–2 additional files. If you are loading more than 4 files, stop and re-route.

---

## Step 1 — Always Load First

**`analysis/voice-profile.md`** — ~3,000 tokens. Contains identity, voice laws, hard refusals, phrase bank, signature tells, golden examples. This is the full calibration in one file. Load it before any copy task, every time, no exceptions.

*Do not load `me.md` + `writing-style.md` in full as a substitute. `voice-profile.md` already contains the essential content of both. Load full `writing-style.md` only for Level 3 tasks.*

---

## Step 2 — Pick Your Calibration Level

| Level | Task type | Files beyond voice-profile.md |
|---|---|---|
| **1** | Caption, pin, subject line, quick edit | None — `voice-profile.md` is enough |
| **2** | Email, carousel, landing page, welcome sequence | voice-profile.md + 1–2 task-specific files from the table below |
| **3** | Sales page, about page, home page, webinar script | voice-profile.md + writing-style.md (specific sections) + analysis file + 1 example |

## What Each File Is Actually For

| File | What it's for | When NOT to load it |
|---|---|---|
| `analysis/voice-profile.md` | AI-ready calibration. Full voice in ~3,000 tokens. Load first, always. | Never skip for copy tasks |
| `analysis/writing-style.md` | 14 sections of deep patterns. Load specific sections by number. Never load in full for Level 1–2. | Captions, pins, subject lines, quick edits |
| `me.md` | Identity anchor. Already covered by voice-profile.md. Use for quick factual lookups only. | Any copy task (use voice-profile.md instead) |
| `analysis/copy-task-routing.md` | Full routing rationale — 545 lines. Load only when COMPASS doesn't answer your question or you need the full reasoning. | Normal sessions where you know the task |
| `analysis/sales-page.md` | Section-by-section BOH sales page map. Load only for sales pages, opt-in pages, or when writing a full CTA catalog. | Emails, carousels, social copy |
| `analysis/soft-sunday.md` | SS-specific dos/don'ts, formatting rules, CTA logic. Always load before any Soft Sunday. | Non-SS tasks |
| `analysis/subject-line.md` | 11 patterns from 351 broadcasts, template banks, Three Jobs decision framework. | Non-subject-line tasks |
| `frameworks/carousel-quickref.md` | Compressed carousel reference for agents. Use only after source examples have already calibrated the task. | Non-carousel tasks; learning Alyssa's carousel style from scratch |
| `frameworks/social-hooks.md` | 7 hook formulas + raw examples. Load when writing or selecting a hook. | Non-hook tasks |
| `analysis/carousel.md` | 9 top-performer post-mortems + performance data + image links. Deep reference only — load when you need the why behind a pattern. | Standard carousel execution |
| `analysis/carousel-slide-by-slide.md` | Full slide-by-slide breakdown of real top carousels. Load when learning the sequence or writing from examples. | Quick carousel execution from an approved brief |
| `examples/Carousels/captions.md` | Real captions, permalinks, and performance fields for the top carousel examples. Load before writing carousel captions or studying hooks. | Non-carousel tasks |
| `analysis/workshop-webinar-analysis.md` | All 3 BOH webinars analyzed. Load for webinar scripts only. | Everything else |
| `frameworks/` | Pattern layer. Load alongside corresponding analysis file — not instead of it. | Without the matching analysis file |
| `examples/` | Real copy. Level 2+ requires at least 1 real example before writing. Read before writing, not after. | Level 1 tasks |
| `programs/back-of-house/` | Course content. Load only when referencing BOH modules by name or writing BOH student-facing content. | General copy tasks |
