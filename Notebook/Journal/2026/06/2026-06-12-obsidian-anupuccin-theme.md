# AnuPpuccin Obsidian Theme

Theme installed and customized for the vault.

- **GitHub:** https://github.com/AnubisNekhet/AnuPpuccin
- **Hub docs:** https://publish.obsidian.md/hub/02+-+Community+Expansions/02.05+All+Community+Expansions/Themes/AnuPpuccin

The CSS file (`annapuchin.md` in Team Inbox, originally `annapuchin.css`) contains the full custom snippet — move to Obsidian snippets folder when applying.

**Modes:** Dark + Light  
**Key customization applied:** File properties visible only on hover, plugins + CSS snippets layered on top of base theme.
