# Meeting Prep Agent

Before any upcoming client meeting, an AI agent runs a quiet preparation pass. It reads the relevant client notes, pulls the last call summary, scans the outstanding to-do list, and surfaces exactly 3 concrete things to push forward.

No digging through folders before the call. The agent does the recall. You show up ready.

This is worth building. It fits naturally into the existing client workflow and would run on a trigger as soon as a meeting lands on the calendar.
