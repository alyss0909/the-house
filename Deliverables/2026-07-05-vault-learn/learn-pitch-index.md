# Content OS — Product Pitch + Teach Index

**How to use this:** Each entry gives you (1) verbatim Identity for CTA-safe naming, (2) 3–6 **teachable snippets** you can deliver free in a carousel/email (the "teach a small thing in a nuanced way + create a gap" move), (3) 2–4 **real pitch angles** quoted from her actual posts/emails, and (4) **the gap it fills** — the pain the free teaching should open that the product closes.

**Chat-word / name discipline (from BUSINESS.md):** Product names and chat words are verbatim or they don't exist. If a chat word is blank, write "manual selection needed" — never invent a placeholder. Never pitch Sunset/Geriatric/Idea-stage products.

**Prices, chat words, and status below are teaching-angle memory aids only — [[Hermes/BUSINESS]] is the only source of truth.** Re-verify every price/chat-word/status against BUSINESS.md's live offer map before any CTA goes out; do not trust the values re-typed in this file.

**The offer-rotation lens (from BOH Phase 2, "The 84 Combinations"):** You never need a new product to keep re-offering. Rotate framings — **feature / result / fresh angle / bonus** — "The dish is the same, you're just describing different notes." Each entry below flags rotation options where the material supports it.
