## The pipeline (one idea in, publish-ready out)

```
SEED (your journal/voice memo/real moment, or a slot that needs filling)
   │
1. THE TEST KITCHEN — the pitch swarm
   4-6 pitchers with FORCED divergent personas (the one proven fix
   for AI homogenization): the Historian, the Contrarian, the Scroller,
   the Obsessive, the Bestie. Each pitches concept + hook + comment word
   + which product/teaching it ladders to. Slot-aware (calendar spec).
   │
2. MEAN ALYSSA — the adversarial gate
   Built from your Brutal Critic prompt + TASTE + ANTI-AI + your verbatim
   rejection quotes from all 5 gate rounds. BLIND scores before any debate
   (no anchoring), sycophancy detection, P0-P3 severity verdicts.
   DECOY CALIBRATION: she periodically gets a known-bad concept slipped in;
   if she praises it, she's benched and rebuilt. We QC the critic itself.
   Survivors advance by TOURNAMENT (compare, don't score in a vacuum).
   │
3. THE MAITRE D' — the algo scorer (honest version)
   NOT a virality predictor (documented snake oil). A weighted rubric
   validated against YOUR real post metrics: DM-shares > saves >
   swipe-through > comments; hook gates everything (Blotato weights it 50%).
   Encodes the mechanical rules: slide 2 = second standalone hook (unswiped
   carousels re-serve opening on slide 2), 6-13 slides, save-bait = reference
   value, share-bait = identity/permission content.
   │
4. THE SOUS CHEF — the teacher
   Pulls the REAL lesson from the product index + BOH teaching indexes:
   one nuanced snippet, taught the way you teach it, gap opened toward
   the slot's matched product. No generic advice survives this station.
   │
5. HERMES — the voice pass
   The clone writes final copy under the full protocol (deep-load, Postmaster
   for emails, 9-slide spine for carousels, ANTI-AI final pass, micro-style
   counts). Six-pass refinement adapted from the best public chain:
   Skeptic → Expert → Scroller → Editor, then the lineup test vs your
   gold standards.
   │
6. THE PASS — batch assembly & handoff
   Output packaged per slot: slides spec + caption + comment word (verified
   in Notion or halted) + Simmer/email variants + pin titles/descriptions
   (7×10×4×5 engine). Delivered to Notion, designer tagged. Stats flow back
   weekly; winners get promoted into the brain.
```

**Voice-drift defense (the documented swarm failure mode):** only Hermes writes final copy; every other agent trades in concepts and critiques. Gold-standard lineup test on every piece. Your corrections keep flowing through the learning loop - the scheduled deep-read routine is already mining one real piece per run and flagging contradictions for you.
