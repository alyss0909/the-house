## The create-viral-content Six-Pass Refinement Protocol — Mapped to Mean Alyssa

Source: `aaaronmiller__create-viral-content | 51483020215d9259c7565511c289d87ac43d3bcb | references/refinement-protocol.md`

Their protocol (five adversarial passes, run in a fixed order, plus a separate thumbnail-refinement track that doesn't apply here):

1. **Skeptic** — "why should I care, what's actually new here" — kills padding/no-insight content
2. **Expert** — "is this technically accurate, what would an expert nitpick" — catches oversimplification and overconfident claims
3. **Scroller** — "would I stop scrolling for this, what's the hook" — the attention gate
4. **Competitor** — "how is this different from the 10 similar posts" — differentiation check
5. **Editor** — "what can I cut without losing meaning" — the tightening pass

Mapped against Mean Alyssa's current gate (blind score → ANTI-AI senior pass → tournament):

- **Skeptic and Editor passes are new structure, not new judgment** — Mean Alyssa's ANTI-AI senior pass already does slop-detection, but doesn't explicitly run a standalone "so what, what's the actual insight" pass BEFORE the anti-AI check. Sequencing the Skeptic pass first (kill content with no insight before spending cycles polishing it) would save Mean Alyssa a round of wasted scoring on concepts that are voice-perfect but substance-empty.
- **The Competitor pass has no current equivalent.** Mean Alyssa's tournament compares surviving concepts against EACH OTHER, but never explicitly asks "how is this different from the last 10 things we published in this format?" This is a genuine gap — recycled-construction detection (her own "never reuse a documented winner" rule) is currently manual pattern-matching by whoever runs the gate; formalizing a Competitor pass that checks new concepts against the actual list of recent real decks/captions would operationalize that rule.
- **Expert pass has a rough equivalent** in Sous Chef (extracting the real BOH lesson) but Sous Chef isn't adversarial — it doesn't ask "would an expert nitpick this teaching claim." Worth considering only if teaching-content errors become a recurring problem; not urgent.
- **Pass order matters and is worth adopting as-is**: Skeptic → Expert → Scroller → Competitor → Editor. Their stated rationale — "no point polishing content that has no value," "hook only matters if content is solid," "differentiation requires strong foundation," "cutting is the final step" — is sound and compatible with Mean Alyssa's existing blind-first discipline. Adopt the ORDER, not their voice/tone language.
