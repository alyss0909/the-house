# First Relay Output Spec

Do not build the full relay yet.

The first practical target is one source-grounded weekly content stack.

## Input

The future relay may accept:

- Alyssa idea.
- Email.
- Obsidian note.
- BOH module.
- Weekly focus/theme.
- Product/freebie target.
- Best-performing post pattern.

For the first test, keep the input small: one idea plus one weekly focus/theme.

## Best case behavior

The relay should be able to take one real Alyssa idea and turn it into a weekly content stack that feels source-grounded, strategic, useful, and recognizably hers.

It should know when an idea wants to be education, when it wants to be offer-led, and when it wants to be For The Girls. It should preserve the monthly rhythm without making the rhythm feel mechanical.

It should deepen a concept with the right amount of BOH/source material, not turn every post into a mini course. The draft should teach one useful thing, create an intentional gap toward the right next step, and avoid beginner/basic explanations unless Alyssa explicitly wants that.

## Scoring expectations

Each draft should be scored against:

- Source-groundedness.
- Alyssa voice/taste.
- Strategy fit.
- CTA accuracy.
- Product/theme match.
- Non-genericness.
- "Would Alyssa still have to wrestle this?" risk.

## Export shape

The eventual weekly output should be able to include:

- Carousel concept.
- Hook.
- Slide-by-slide breakdown.
- Slide copy.
- Caption.
- First comment.
- CTA and exact ManyChat/chat word, or `manual selection needed`.
- SEO/keywords where relevant.
- Designer handoff notes.
- Source citations and trust-tier labels.

## Suggested local relay roles later

This is a proposed future shape, not an implementation request.

| Role | Job |
|---|---|
| Source Retriever | Finds Alyssa-original notes, BOH lessons, emails, carousels, and product facts. |
| Strategy Matcher | Chooses theme, slot, offer/freebie, CTA lane, and post type. |
| BOH Teaching Extractor | Pulls one nuanced teaching point from the most relevant module. |
| Concept Architect | Builds the idea arc and decides whether a copy framework is useful. |
| Wren / Alyssa Voice Writer | Writes copy in Alyssa's voice from retrieved evidence. |
| CTA Mapper | Checks product/freebie/offer fit and exact Notion chat word. |
| Quality Scorer | Scores source truth, nuance, voice, offer fit, usefulness, and non-genericness. |
| Exporter | Formats for Notion, Canva, and designer handoff later. |
