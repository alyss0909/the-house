# Review the suggested Gmail filters and give the go

## What this is

You asked to have the filter file surfaced as a task so it is easy to find. There are two versions of the same plan, both in the Mailroom desk:

- [[gmail-filters-paste-and-go]] — the ready-to-run version. Every "From" string is split by which of your four inboxes it belongs to, verified against real mail. You paste each one into that account's Gmail filter UI yourself. This one is yours to run whenever, no waiting on Larry.
- [[filter-plan]] — the full plan, still marked "awaiting your go" for the filters Larry would build for you through Composio instead of you pasting them by hand.

Both do the same three things: skip-inbox for recurring social and event-reminder noise, a "Tool updates" label for tool marketing you sometimes read, and mark-as-read for system notices. Nothing deletes mail. Security alerts, failed payments, Dennis, real humans, and your own broadcasts are all explicitly never filtered.

## The decision in front of you

One thing to settle first: filters or the bulk-archive habit. You said you might just archive everything going forward and keep an easier flow, which is what we did for personal and support promotions. If that is the direction, most of these skip-inbox filters become unnecessary and only the "Tool updates" label and the security-never-touch list still matter. Worth deciding which path before building anything.

- [ ] If filters: run (or hand Larry the go for) the skip-inbox and Tool-updates rules
