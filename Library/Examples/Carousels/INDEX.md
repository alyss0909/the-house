## How To Use These

**For voice calibration:** Open the cover slide + 2-3 interior slides from any carousel in the same archetype you are writing. Read the copy. Notice the layering of all-caps body text, display serif headline, and handwritten annotation.

**For structure calibration:** Pair the slide images with [[Studio/Analysis/carousel-slide-by-slide|carousel-slide-by-slide.md]]. It explains each slide's role, tactic, visual metaphor, and voice note.

**For framework reference:** [[Studio/Analysis/carousel|Studio/Analysis/carousel.md]] maps the larger patterns. Do not read images without the analysis alongside.

**Rule from Library/Examples/INDEX.md:** Always read at least one real example in slide order before writing. Never start cold.
