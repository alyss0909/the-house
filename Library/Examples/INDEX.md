## How a team member uses this

- Before writing an email: read 2-3 examples from `simmers/` or `soft-sundays/` to calibrate voice
- Before writing a sales page: read `sales-pages/` for real structure + language
- Subject lines: start from `subjects-and-previews.md` — don't blank-page it
- Carousels: `carousels/` shows how she opens, builds, and closes a slide deck

**Rule:** Always read at least one real example before writing. Never start cold.
