## How to use these

**For voice calibration:** Read the pep talk section opening of any email here before drafting a Soft Sunday. Notice the big-sister tone, the casual lowercase, the permission-slip energy.

**For format reference:** Soft Sundays are structured with recurring sections — The Pep Talk, The House Report, Nerding Out On, Bake Break, Creep My List, Time Luxuries, The Check In. Not all sections appear in every issue.

**Rule from Library/Examples/INDEX.md:** Always read at least one real example before writing. Never start cold.
