## Criteria for Organic Marketing

My goal is not to swap one platform for another that we have to constantly be managing and coming up with new ideas for. It's to reimagine this marketing strategy altogether.

**My rules for organic marketing to replace daily posting:**

- Create once, work forever
- Doesn't need new content to stay "relevant"
- Content is actually interesting to buyers
- Attracts buyers and can pre-qualify people before they get on my list
- Prioritizes quality content and evergreen content over excessively needing new stuff
- Can be batched and automated easily
- Shouldn't need to be checked on, tweaked, or optimized based on trends

---

## Why Pinterest Checks All the Boxes

**Once a pin is indexed, it's findable forever.** Our job is just to keep resurfacing that pin.

Pinterest likes fresh pins but doesn't say anything about creating new content. We can keep resurfacing your best content in fresh packaging.

Every pin we put out is funnel-forward. We're focusing on well-written content designed to build your authority. Every person who comes to your website from Pinterest has already been pre-qualified through your content.

Can be batched — we're about to batch six months of content. And it really isn't trend-based. In the last 10 years I've been on Pinterest, I think there've been two major algorithmic changes. Everything else has been pretty minor. Search intent is stable and predictable.

## Pinterest Best Practices (The Rules)

**Know these before you batch — this plan is set up to never break them, but you need to know why.**

1. **Maximum 5 boards per URL** — We max that out by using 5 super relevant boards and sending all 7 URLs to those 5 boards, spaced out.

2. **7–10 day spacing** — Before reposting the same URL to the same board. Our rotation logic makes sure that never happens.

3. **Post consistently** — Pinterest loves regular users. Less of a rule, more of a best practice.

4. **Freshness first** — The golden rule. All Pinterest cares about is fresh pins.

5. **Quality images and keyword-rich metadata** — Good designs + keywords everywhere.

6. **No duplicates** — Same image + title + board + URL = flagged. This is the biggest one. This is exactly why we make so many fresh pins.

## What "Fresh Pin" Actually Means

Pinterest's own criteria for freshness: unique combinations of your **image**, **title**, **description**, **URL**, and **board** in search results.

When it sees a new combination, it gives it a little visibility boost.

**The metadata elements (listed by weight):**
- New destination URL → Pinterest values this most highly
- New image
- New pin title
- New pin description
- New file name
- New alt text
- New board

Not all metadata has equal weight. A new URL + image is valued much higher than just a new file name.

**What counts as fresh:** Take the same URL with a new image and pin it to a new board = absolutely fresh. Take the same image and title, repin to the same board = not fresh.

**What doesn't work:** Putting the same thing on every board over and over. What does work: mixing and matching combinations — like a capsule wardrobe — to create hundreds of unique combinations.

## The Master Plan: 7×10×4×5

**What you need:**
- 7 URLs (4 blogs + freebie + tripwire + offer)
- 5 design templates
- 2 overlay text options per URL
- 4 title and description combos per URL
- 5 relevant boards

**How the math works:**

For each URL:
- 5 design templates × 2 overlay text options = **10 designs**
- 10 designs × 4 title/description combos × 5 boards = **200 unique fresh pins**

For all 7 URLs:
- 200 × 7 = **1,400 unique pins**

That's more than enough to batch and automate pins going out every day for 6 months, never repeating the same URL + board combo within 7 days, fully Pinterest compliant.

---

**Example of overlay text pairs (Blog: "4 Steps to Build a Trustworthy Brand as a Nutritionist"):**
- Overlay Text 1: "How to design a nutrition brand that clients trust instantly"
- Overlay Text 2: "The secret branding formula wellness coaches swear by"

Both lead to the same blog. Both leverage keywords. Slightly different angle.

**Example of title/description combos (same blog):**
- Title 1: "4 Powerful Brand Strategies for Nutritionists Who Want Trust"
- Description 1: "Struggling to build client trust as a nutritionist? These 4 strategies will help you create a trustworthy, memorable brand perfect for wellness entrepreneurs."
- Title 2, Description 2 → different wording, same URL, same keywords

## Pre-Design Checklist

Before you open Canva, make sure your pin planner spreadsheet has:

- [ ] 4 clear signature topics with at least one seed keyword each
- [ ] At least 5 long-tail phrases researched per pillar
- [ ] One seed keyword assigned to each of your 5 boards
- [ ] Board titles written exactly as they appear in Pinterest (the planner uses these)
- [ ] All 7 URLs linked and correct
- [ ] Overlay text options written (2 per URL, 7–10 words each — never go over 10)
- [ ] Most relevant board assigned for each URL (your blog posts probably match one board best)
- [ ] 4 title + description combos written per URL (title under 100 chars, description under 500)
- [ ] File naming convention decided (e.g., "branding-tips-for-nutritionists-pin1-Q1")

**For file names:** We use the keyword + pin number + quarter. It's another layer of metadata that Pinterest reads. It's two seconds of work and it's worth it.

## Pin Design Styles

Five main types — choose five you love, make them yours.

**1. Mockup pins:**
Especially effective for freebies and offers. We actually use mockup pins for all of our content because our blog posts all have content upgrades. Works well across everything.

**2. Image overlay text pins (the classic):**
A background image + text overlay. Most common. This is what I use most often.

**3. Quote pins:**
Just text over a gradient or plain background. Doing really well on Pinterest right now. Simple, clean, readable.

**4. Video pins:**
Pinterest went hard on these for a while. Now image vs. video doesn't seem to matter much — stick to images unless your main content is video. Or just swap the image out for a video in a pin you've already designed.

**5. Infographic pins:**
Pinterest really loves these. Different format — batched differently (see below). Some accounts grow almost entirely with just infographics. Worth including in your mix.

---

**How to choose:** Go to Canva Design Trends — they share what's trending in design every year. Pinterest has similar trend data. Look at what's working right now, adapt to your brand, use that as your starting point. But if you want to be super simple: just snag five template styles and go.

**Template preparation:**
- Change to your brand colors + font palette
- Add your go-to call to action (e.g., "Read more" / "Get the free guide")
- Add your website URL if you want
- On mockup pins, add your mockups
- Make sure there's room for 7–10 words in the overlay text area (test with placeholder text)
- Set your text box anchor (bottom for text that grows upward, center for text that stays centered)

## Bulk Creating in Canva

**Steps:**

1. Choose your 5 design templates in Canva — put them all in one design file
2. Prepare each template: brand colors, CTA, URL, anchor settings for text
3. Rename each file before bulk upload using the file name field (click pencil → type keyword + pin number + quarter)
4. Go to **Bulk Create** (sidebar in Canva) → enter data manually or upload a sheet
5. Clear the table → paste in all 14 overlay text options (2 per URL × 7 URLs)
6. Connect your data: click the overlay text box on your template → "Connect data" → "Overlay text"
7. Click **Continue** → Canva generates all 70 designs (14 overlay texts × 5 templates)
8. Do quick tweaks: swap background images, adjust spacing, fix anything that looks off
9. For pins that don't need changes: leave them
10. **Download** → File → Download → PNG → **2×** (exports high-quality PNGs — Pinterest loves this)

**Quick batch tip:** When you go through the 70 designs, change a color on some of them (bottom bar color, button color). Small differences make your designs look distinct even when using the same template. Not required for every pin, but worth doing on a few.

**Advanced setting:** In Bulk Create → Advanced Options → Group designs by data → When enabled, all versions of one overlay text are grouped together and named after that overlay text automatically. This auto-names your files for you. Useful but then you lose the ability to edit all templates at once.

## Infographic Pins

Pinterest loves infographics. These batch slightly differently because they need a headline + 5 individual points.

**Approach:**
1. Decide on a template and a set number of points (5 works well — consistent across all infographics)
2. Use the Pin Keymaster bot to generate an infographic for each of your 7 URLs: give it your blog post and ask for one headline + five points
3. In Bulk Create → paste in: headline + all 5 points
4. Connect each data field to the right element in your template (this is more finicky — you have to connect 6 things instead of 1)
5. Generate → tweak designs → download

**Format tip:** Infographics are worth the extra setup. Some accounts grow almost entirely using them. Just decide upfront on your template style and how many points per infographic so they stay consistent and recognizable.

## Download and Export

- Download as **PNG** (not JPEG)
- Choose **2×** resolution (Pinterest loves high-quality images)
- All 70 designs download as one zip folder with individual file names
- Extract the zip → upload to a Google Drive folder
- Make the folder shareable (Anyone with the link can view) — this is required for the scheduler
