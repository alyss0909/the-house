## Pre-Design Checklist

Use your pin planner to…

- [ ] Define 4 Signature Pillars for your niche (core content themes).
- [ ] Assign a Seed Keyword (1–2 word root keyword) to each Pillar.
- [ ] Research Longtail Keywords (specific, searchable phrases) for each Pillar.
- [ ] Create 5 Pinterest Boards:
  - [ ] Assign a Seed Keyword to each Board.
  - [ ] Give each Board an Internal Name (simple, one word).
  - [ ] Write a Board Title (optimized, public-facing, keyword-rich).
- [ ] Write Blog Titles and Overlay Text:
  - 1 overlay text + 1 alt overlay text per blog.
  - Make sure longtail keywords are woven in naturally.
- [ ] Assign Each Blog to the Most Relevant Board (match pillar/keyword to board).
- [ ] Write 4 Pin Titles + 4 Pin Descriptions per URL:
  - Follow the Pinterest-friendly template.
  - Sprinkle in seed and long tail keywords.
- [ ] Create SEO Friendly File Names:
  - Use Seed Keyword + Pin # + Quarter (e.g., branding-pin1-Q1).

## Design Checklist

Use Canva and the curated templates to…

- [ ] **Choose 5 Design Templates** — Pick layouts optimized for Pinterest (2:3 ratio). Use Canva Design Trends for inspo.
- [ ] **Add Branding Elements** — Apply your brand fonts and color palette. Add your go-to CTA. Insert your website URL. Add mockups or product images if relevant.
- [ ] **Format them** until you think they'll fit 7–12 word overlays.
- [ ] **Update File Names** — Rename each Canva file BEFORE bulk upload. Format: `[seed-keyword]-pin#-Q1` (ex: branding-pin1-Q1).
- [ ] **Prepare Your Data Sheet** — Fill in headlines and points (one row per design). Make sure there are no empty cells (Canva will choke).
- [ ] **Import Data into Canva (Bulk Upload)** — Open Bulk Create > Upload your spreadsheet.
- [ ] **Connect the Data to the Design** — Link overlay text to overlay field. Link point 1 to point 1 field, and so on (infographics).
- [ ] **Bulk Create Your Designs** — Click "Continue" > "Generate all." Wait for Canva to create all your pins.
- [ ] **Review and Make Small Edits** — Adjust text that doesn't fit. Tweak spacing, font sizes, image placements where needed.
- [ ] **Export Designs** — Export at 2x for high-quality PNGs.

## Pin Planner: Step-by-Step

### Step 1: Open the Spreadsheet

Navigate to your PIN PLANNER spreadsheet.

Approve the script when prompted (Tools > Script Editor > Authorize or follow any pop-up permissions).

> [!note] About the Permissions Screen
> Google will show you a very dramatic permissions screen when you run this script — it says the app can 'see and delete all your Google Sheets.' That's because the script lives inside YOUR spreadsheet and Google doesn't allow us to narrow the language. Don't worry: Your Planner data is processed only within your own Google account's Apps Script runtime, never transmitted to our servers or stored by us. It only runs in YOUR sheet, and we don't see, collect, or access anything (wouldn't know how if i wanted to lol!). You can also turn it off anytime in your Google settings. xx

1. You'll see this when you copy the sheet
2. You'll see this when you use the Lounge Menu
3. Click Advanced → Go to my App & click allow

---

### Step 2: Finalize the Keyword Planner Tab

- [ ] Confirm all 5 Pinterest board names are listed and match your Pinterest exactly.
- [ ] Add a seed keyword for each board.

---

### Step 3: Map Your Boards + Assets

- [ ] Confirm your 7 URLs are entered correctly.

---

### Step 4: Upload Your Canva Images to Google Drive

- [ ] Download all 70 images from Canva as a ZIP file.
- [ ] Extract the ZIP.
- [ ] Upload all 70 images to a Google Drive folder.
- [ ] Change folder sharing settings to "Anyone with the link can view."

---

### Step 5: Add to Pin Planner

- [ ] Switch Drive to list view.
- [ ] Sort by name.
- [ ] Select all images and copy the links.
- [ ] Paste them into a Notepad to clean formatting.
- [ ] Paste the cleaned links into the Pin Images tab, column L.

---

### Step 6: Assign Each Image to an Asset

- [ ] For each image, select which asset (URL) it belongs to using the dropdown.
- [ ] Ensure you assign exactly 10 images per asset.

---

### Step 7: Finalize the Pin Copy Tab

- [ ] Write 4 titles and 4 descriptions for each of your 7 URLs.
- [ ] Keep titles under 100 characters.
- [ ] Keep descriptions under 500 characters.
- [ ] Use 5+ longtail keywords in each description.

---

### Step 8: Confirm Batch Prepping Details

Go to the Batch Prepping tab. Confirm:

- [ ] All boards are entered with exact names.
- [ ] Each board has a seed keyword.
- [ ] All image links are live (check in incognito window).
- [ ] Every image is mapped to a URL.
- [ ] All copy is filled in and properly formatted.

---

### Step 9: Select Your Scheduling Start Date

- [ ] Choose a future start date for your schedule from the Batch Prep tab.

---

### Step 10: Generate Your Pin Schedule

- [ ] Open the Pinterest Lounge menu.
- [ ] Click "Generate Pins."
- [ ] Wait for the app to process all 1,400 pins.

---

### Step 11: Check for Errors (If Any)

- [ ] Use the error report in the sidebar to see what's missing.
- [ ] Edit any red-highlighted cells.
- [ ] Fix issues in source tabs and regenerate schedule.

---

### Step 12: Prep for Notion Social or Pinterest Bulk Upload

**If using Notion Social:**
- [ ] Sign up for Notion Social
- [ ] Download Schedule tab as CSV.
- [ ] Upload to Notion Database (Import > CSV)
- [ ] Match properties: Make sure Media comes in as TEXT or FILE property (not URL)
- [ ] Connect your Pinterest to Notion Social
- [ ] Once in Notion: Be sure the MEDIA column is a FILE property
- [ ] Add your Pinterest to "platform" on all rows in Notion database
- [ ] You're All Set!

**If using Pinterest Bulk Upload:**
- [ ] Click "Prep for Pinterest" in the custom menu.
- [ ] Script will scrub images + chunk into 200-pin sheets.
- [ ] Download each sheet as CSV.
- [ ] Upload directly to Pinterest in batches.

---

### Step 13: Maintenance

- Check Pinterest analytics monthly.
- Watch for trends in saves, clicks, and designs.
- Note high-performing overlays or topics.

> 🍾 YOU'RE DONE. Sit back and let the Back of House engine compound every post, every click, every save, every day for 6 months straight.
