# Programs — Index

> Course content and flagship program material. Module indexes, workbooks, video transcripts, dashboards, slides. The deep knowledge layer.

---

## What lives here

One subfolder per program. Each contains:
- A program-level index (phases, modules, what's covered)
- Associated workbooks (as markdown)
- Video transcripts (text only — no binary files)
- Notion dashboard content
- Slides text layer (extracted from Canva/Drive)
- Sales page copy for that program

## How to add a new program

1. Create a subfolder: `Library/Programs/[program-slug]/`
2. Create `INDEX.md` inside it — list every module/section with a one-line description
3. Add content files one at a time (transcripts, workbooks, dashboards)
4. Update the table above
5. Update the master [[INDEX]] checklist
